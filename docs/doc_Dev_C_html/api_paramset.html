参数设置
ParamSet 用于设置读写器参数，可以在读写器实例句柄的生命周期内随时调用此函数。
一旦设置了参数，它将在读写器的整个生命周期中起作用，除非它被修改。读写器上电时会恢复为默认值。如需要永久保存，即重新上电后也能保存参数则通过修改 默认配置来实现。

过滤器设置

UHF读写器基本标签操作读，写，锁，销都支持过滤操作，可以设置一个过滤规则或者多个过滤规则。单过滤规则参数关键字为 MTR_PARAM_TAG_FILTER； 多过滤规则参数关键字 为MTR_PARAM_TAG_MULTISELECTORS。
