错误处理
 
所有具有返回值的API函数将在成功时返回MT_OK_ERR。如果返回值为MT_IO_ERR，则表示网络连接或串口连接有问题， 应首先调用CloseReader 并检查这些连接，然后再尝试调用InitReader_Notype。 至于MT_CMD_FAILED_ERR，它只是意味着函数失败，它不是致命错误，你可以接下来做任何操作。但是若在盘点中出现，一般为通信数据异常。出现频率较多则意味通信状况较差。至于MT_CMD_NO_TAG_ERR，严格来说它不能被称为错误，表示标签操作时没有发现标签。
  
以MT_HARDWARE_ALERT_ERR_BY开头的错误属于严重错误，不容忽视，有可能导致硬件损坏。有一些不正确的操作会导致这些错误，这些操作包括：
  • 在没有接天线的读写器端口进行标签操作
  • 使用不合格的天线
  • 环境温度过高
  • 天线前方离金属很近，会造成回波损耗过大
最好先关闭读写器并检查发生这些错误时的工作条件和工作环境。
 
还有另一种方法可以进行错误故障排查，您可以在发生错误后立即调用GetLastDetailError函数， 以获取更详细的错误代码和由字符串表示的错误消息。

对于GetLastDetailError接口调用，应遵循以下几点规则。

  • InitReader_Notype返回失败后，由于句柄hreader无效，故不应调用GetLastDetailError接口
  • RFID操作例如，读写锁，参数配置等接口返回失败后，即非MT_OK_ERR后可以调用GetLastDetailError接口
  • 在回调函数ReadingErrorHandler处理函数中可以调用GetLastDetailError接口
    •  
    当前版本SDK的所有函数（InitReader_Notype函数除外）对于同一个读写器实例句柄都不是线程安全的。 用户应确保API函数调用没有竞争条件或保证多线程间互斥调用API函数。对于不同的读写器实例句柄，没有上述限制。
        

    以下为调用GetLastDetailError函数返回读写器的错误代码表。列表中的代码由读写器返回，详细可参考表描述。

    Status Code(HEX) Status Code(DEC) 描述与处理
    0x0000 0 成功
    0x0100 256 发送Data Length 错误，可能读写器固件版本与API版本不匹配，检查相关版本
    0x0101 257 命令不可用，应先确定读写器支持此命令；其次应检查读写器状态转变， 若发生从应用状态自动转为启动状态，常因读写器供电不稳定导致软复位。
    0x0105 261 参数值不可用，检查相关参数值
    0x010A 266 波特率不可用
    0x010B 267 当前工作频段不可用
    0x0200 512 App Firmware层程序CRC不正确，可以尝试重新升级读写器固件
    0x0302 770 AFlash未定义错误，闪存写入失败
    0x0400 1024 未发现标签，常见于功率不足或者过滤不匹配
    0x0402 1026 协议不可用，应检查读写器固件，API是否支持当前协议
    0x0404 1028 启用了写入后读出功能时，写入成功但读失败。可重新执行
    0x040A 1034 常规标签错误（读/写锁定，kill命令），可重新执行
    0x040B 1035 读取存储区域超出限制的长度（例如，一次只能读取96块）
    0x040C 1036 不可用的销毁密码，常见未写销毁密码。重新写销毁密码
    0x0420 1056 GEN2协议错误
    0x0423 1059 存储区越界，常见传入错误的长度，超出了标签存储范围
    0x0424 1060 操作的存储区已被锁定，可尝试设置密码
    0x042B 1067 能量不足，可调大功率重试
    0x042F 1071 非特定错误
    0x0430 1072 未知错误
    0x0500 1280 频率值不可用
    0x0504 1284 温度过高
    0x0505 1285 回波损耗过大，常见于金属反射的环境，也较少发生在频率，天线等原因

    0x500F 20495 宜链温度标签检测温度超出量程
    0x50FF 20735 宜链温度标签检测温度不稳定

    0xAA03 43523 读OEM寄存器失败
    0xAA04 43524 该命令执行失败
    0xAA2A 43562 OEM格式化失败
    0xAA31 43569 载波开启或关闭操作失败(未接天线)，需要接天线或频谱仪
    0xAA40 43584 表示保存配置命令写OEM寄存器失败
    0xAA4A 43594 驻波检测失败，可能需要检查硬件问题
    0xAA4B 43595 表示复位或上下电等操作失败

    0xAA4D 43597 表示设置SESSION2 /SESSION3 TARGET 为A失败
    0xAA55 43605 用户自定义存储区命令，读写执行失败
    0xAA56 43606 用户自定义命令配置或读取执行失败
    0xEE01 60929 芯片未执行到应用层错误
    0xEE02 60930 芯片固件启动升级失败
    0xEE03 60931 芯片固件继续升级失败
    0xEE04 60932 芯片固件结束升级失败
    0xFF11 65297 FLASH初始化失败
    0xFF12 65298 GPIO初始化失败
    0xFF13 65299 定时器初始化失败
    0xFF14 65300 SPI初始化失败
