参数
ParamGet和ParamSet函数用于获取和设置读写器的参数。 读写器的每个参数都由一个键值表示，该键值是Mtr_Param枚举类型。参数可分为只读和可读写参数两种。 读写器的参数将一直有效，直到将其设置为其它值。
    

参考以下常用参数列表，关键字KEY字体红色表示参数只读，蓝色表示可需读写。 关键字KEY背景黄色表示保留/预留/暂不支持/已过时，橙色背景表示特殊用途。
