◆ CParamlen

int CustomParam_ST::CParamlen

为pCusParam 的字节长度,注意，当pCusParam为传入参数时，CParamlen仅仅为50字节之后为具体参数值的长度； 当pCusParam为传出参数时，CParamlen包含为参数名前50字节的长度。

◆ pCusParam

void* CustomParam_ST::pCusParam

包含了自定义扩展参数关键字，参数值。 以下3种情况首字节为关键字，决定自定义操作:

  • 首字节为0x01 表示进行载波测试,参数值字节序列为：开启还是关闭(1字节，1开启，0关闭)+天线id(1字节)+功率(2字节)+频率(4字节)。该参数只设置。
  • 首字节为0x02 表示进行切换模块至BootLoader层。无参数值。该参数只设置。
  • 首字节为0x03 表示读写寄存器，参数值字节序列为：4字节地址+4字节数值。该参数可设置获取。
    除上述情况之外，前50字节为参数名，50字节之后为具体参数值，现支持以下关键字：
  • 参数名为"reader/rdrdetails",表示获取读写器硬件信息，返回参数值字节序列:
    16字节预留+4字节硬件版本+12字节序列号. 故传入的pCusParam的CParamlen至少不能小于50+32 字节。
  • 参数名为"reader/moduleinfo"表示获取读写器详细信息，返回参数值字节序列:
    16字节预留+16字节版本信息(4字节Bootloader+4字节硬件版本+4字节固件日期+4字节固件版本)+12字节序列号. 故传入的pCusParam的CParamlen至少不能小于50+44 字节。
  • 参数名为"reader/macaddr"，表示获取mac地址，返回参数值6字节。
  • 参数名为"reader/infacthardwareinfo"，表示获取内部硬件信息，现只有SFM系列支持。返回参数值字节序列:1字节指示是否存在内部信息+64字节信息。
  • 参数名为"Reader/Ex10fastmode",表示普通快速模式与Ex快速模式切换。参数值22字节，字节序列为：模式(1字节，1为Ex快速模式，0为普通快速模式)+1字节(固定值为20)+ 场景1字节(0 为标签密集柜子场景,1 为少量标签场景)+19字节(现全0)。该参数可设置获取。
  • 参数名为"tagcustomcmd/fastid"，表示是否开启fastid功能，需要标签支持。参数值的字节序列：开启还是关闭(1字节，1开启，0关闭)。该参数可设置获取。
  • 参数名为"tagcustomcmd/tagfocus"，表示是否开启tagfocus功能。参数值的字节序列：开启还是关闭(1字节，1开启，0关闭)。该参数可设置获取。
  • 参数名为"gen2op/multiembeddeddata"，表示设置多附加数据项,多个附加数据项总块数不能超过56。参数值的字节序列:附加数据项个数(1字节)+密码(4字节)+附加1bank(1字节)+ 附加1地址(4字节)+附加1块数(1字节)+附加2bank(1字节)+附加2地址(4字节)+附加2块数(1字节)+附加3块数(1字节)+...。
  • 参数名为"gen2op/settag2selected"，保留
  • 参数名为"blf/asyncinvparams"，保留
  • 参数名为"rfcommon/anthoptab"，保留
  • 参数名为"rfcommon/hopmode"，保留
  • 参数名为"Reader/Savestandby"，表示是否在关闭读写器时候将其切换至Bootloader层。参数值1字节，字节序列为：开启还是关闭(1字节，1开启，0关闭)。该参数可读写。
  • 参数名为"R2000/oemregister"，表示读写OEM寄存器，参数值4字节，字节序列为：4字节地址+4字节数值。该参数可设置获取。
  • 参数名为"Reader/realtimeinfo"，保留
  • 参数名为"Reader/impinjfastmode"，预留
  • 参数名为"Reader/dutycycle"，设置占空比参数,参数值4字节。字节序列为:2字节占空比执行前盘点时间单位毫秒+2字节占空比执行的基准时间单位毫秒。
  • 参数名为"Reader/powermode",设置读写器功耗模式,参数值1字节：1字节模式;0x00 FULL Power， 0001 MIN_SAVING, 0x02 MED_SAVING, 0x03 MAX_SAVING。
  • 参数名为"Reader/rssifilter"，读写器内部过滤RSSI，参数值4字节，字节序列为：获取参数时，4字节全0;设置参数为取消内部过滤时，1字节0x01(设置)+3字节全0;
    设置 为启用内部过滤时：1字节0x01+1字节0xAA+1字节有符号RSSI值+1字节0x00。
