类成员变量说明

◆ FastReadDutyRation

* unsigned char BackReadOption::FastReadDutyRation

高4位必须为0，低4位有效，低4位用于表示在读写器盘存期间待机所花费的时间比例。 如果FastReadDutyRation为0-10，则比例为FastReadDutyRation * 5/％。 如果基于Ex芯片的读写器而且FastReadDutyRation为11，12，13,14,15时分别为55，60，70，80，90。

◆ GpiTrigger

GPITrigger BackReadOption::GpiTrigger

如果IsGPITrigger为1，则使用GPI触发条件来启动或停止异步盘存。

◆ IsFastRead

int BackReadOption::IsFastRead

如果为1，则使用高速异步盘存模式，否则使用普通异步库存模式。

◆ IsGPITrigger

int BackReadOption::IsGPITrigger

如果为1，则使用GPI触发条件来启动或停止异步盘存。

◆ ReadDuration

unsigned short BackReadOption::ReadDuration

使用普通异步盘存模式时的盘存周期（以毫秒为单位）。如果使用高速异步库存模式，则该成员无意义。

◆ ReadInterval

unsigned int BackReadOption::ReadInterval

使用普通异步盘存模式时，两个盘存周期之间的空闲间隔（以毫秒为单位）。 如果使用高速异步盘存模式，则该成员无意义

◆ TMFlags

TagMetaFlags BackReadOption::TMFlags

指定使用高速异步盘存模式时标签携带的元数据信息。 如果使用普通异步库存模式，则该成员无意义。
