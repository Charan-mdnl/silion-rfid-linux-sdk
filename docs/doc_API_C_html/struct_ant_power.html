单个天线的功率参数

单个天线功率配置信息，为AntPowerConf中的数组元素

类成员变量说明

◆ antid

int AntPower::antid

天线ID，从1开始编号

◆ readPower

unsigned short AntPower::readPower

当读写器执行读取相关操作时的发射功率，以centi-dbm为单位.例如：设置30dbm，readPower=3000。 注意，通常情况是相同值readPower和writePower，读操作要比写操作距离远。

◆ writePower

unsigned short AntPower::writePower

当读写器执行写入相关操作时的发射功率，以centi-dbm为单位.例如：设置30dbm，writePower=3000。 注意，通常情况是相同值readPower和writePower，读操作要比写操作距离远。
