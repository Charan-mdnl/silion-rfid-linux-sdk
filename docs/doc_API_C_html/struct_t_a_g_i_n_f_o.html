详细描述

用于表示盘存到的标签的详细信息

表示读到的一个标签的全部信息

注解
如果是快速模式盘点，则需要指定返回标签信息项，普通模式默认全部返回

unsigned char TAGINFO::CRC[2]

EPC bank的CRC段，如果E系列返回相位时，此项为初始相位 相位数据转换为度的方式为： 低12位(Bits)的值/4096*360(度)
