GpiTrigger_Type GPITrigger::GpiTriggerType

Trigger type When the trigger type includes _TIMEOUTSTOP, this value is the timeout stop time unit in seconds after triggering. Otherwise, this value is the reverse timeout unit in milliseconds. Reverse refers to restoring the triggered state to a non triggered state. Therefore, triggering with the type of _TIMEOUTSTOP only requires one state transition; Otherwise, two state transitions are required
