◆ FastReadDutyRation

unsigned char BackReadOption::FastReadDutyRation

The high 4 bits must be 0, the low 4 bits are valid. The low 4 bits represent the proportion of time spent on standby during the reader's inventory. If FastReadDutyRation is 0-10, the proportion is FastReadDutyRation * 5%. If FastReadDutyRation is 11,12,13,14,15, the proportion is 55%,60%,70%,80%,90%.
