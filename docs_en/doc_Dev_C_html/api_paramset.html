Parameter Settings
The ParamSet function is used to configure the parameters of the reader. This function can be called at any time during the lifecycle of the reader instance handle.
Once a parameter is set, it remains effective throughout the entire operation of the reader until it is modified again. When the reader is powered on, all parameters revert to their default values. To achieve permanent parameter preservation (where parameters remain effective after a power cycle), modifications must be made to the default configuration.

Filter Settings

UHF reader supports filtering operations for basic tag operations such as reading, writing, locking, and locking, and can set one or multiple filtering rules. The parameter keyword for a single filtering rule is MTR_PARAM_TAG_FILTER; the parameter keyword for multiple filtering rules is MTR_PARAM_TAG_MULTISELECTORS.
